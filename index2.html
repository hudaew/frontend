A hidden <input type="radio"> is linked to the visible label via for/id. When clicked, the label toggles the input's :checked state. A CSS sibling selector input:checked ~ .panel-body then changes the grid row size — no JavaScript needed.

Radio inputs share a name group, so selecting one automatically deselects all others. This enforces the "only one panel open at a time" behaviour without any JavaScript — the browser handles mutual exclusion natively.
